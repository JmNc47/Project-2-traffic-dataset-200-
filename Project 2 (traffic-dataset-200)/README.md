# 📌 CVAT Data Labeling Project

## 📖 Description

This project consists of annotating 200 images using CVAT for object detection tasks.

## 🏷️ Labels

The dataset includes the following classes:

* Person
* Car
* Motorcycle

## 🛠️ Tools Used

* CVAT (Computer Vision Annotation Tool)

## 📊 Dataset Details

* Total images: 200
* Annotation type: Bounding Boxes
* Format: COCO
