# 📌 CVAT Data Labeling Project

## 📖 Description

This repository contains a dataset of 200 images annotated using CVAT for object detection tasks. The project focuses on accurately labeling multiple objects within each image using bounding boxes.

## 🏷️ Classes

The dataset includes the following object categories:

* Person
* Car
* Motorcycle

## 🛠️ Tools Used

* CVAT (Computer Vision Annotation Tool)

## 📊 Dataset Information

* Total images: 200
* Annotation type: Bounding Boxes
* Format: COCO

## 🖼️ Preview

Sample annotated images are available in the **preview/** folder.

## 🎯 Objective

The purpose of this project is to practice and demonstrate skills in data labeling and computer vision dataset creation.

## 🚀 Skills Demonstrated

* Image annotation
* Object detection labeling
